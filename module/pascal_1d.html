Pascal_1D is a submodule used to generate arrays of coeffiecents used for developing finite element basis functions in 1D. Finite element basis functions are defined at internal nodes and used to iterpolate some value between those nodes.

calculate the coeffiecents associated with a univarate Lagrangian polynomial

\[ 1, x, x^2, ..., x^N \]
